# 拉格朗日点 - 实验报告

## 1. 实验目的简述

本实验旨在通过数值方法求解地球-月球系统中L1拉格朗日点的位置，具体目标包括：
- 理解拉格朗日点的物理意义及其在天体力学中的重要性
- 推导L1点位置满足的方程
- 实现并比较不同的数值求根方法（牛顿法、弦截法）
- 分析数值方法的收敛性和精度

_通过求解地月系统L1点的力平衡方程确定其位置。L1点方程由地球引力、月球引力和离心力的平衡推导得出：地球引力项为GM/r²，月球引力项为Gm/(R−r)²（方向与地球引力相反），离心力项为ω²r（假设旋转中心近似为地球）。三力平衡时，方程形式为GM/r² - Gm/(R−r)² - ω²r = 0。

牛顿法利用函数及其导数迭代求解：每次迭代计算当前点的函数值f(r)和导数值f’(r)，通过r_new = r - f(r)/f’(r)更新解，直至收敛（函数值或步长小于容差）。弦截法则采用两点间的割线逼近根，迭代公式为x2 = x1 - f(x1)*(x1-x0)/(f(x1)-f(x0))，无需导数信息。

初始值选择基于物理预估：牛顿法初值3.5e8 m（约地月距离的90%）和弦截法区间[3.2e8,3.7e8]均参考L1点位于地球与月球间约85%地月距离的经验值，并通过绘制函数图像验证零点存在性。数值实验表明，两种方法均能快速收敛至约3.264e8 m，与SciPy结果一致。）_

### 4.1 L1点的物理意义

请分析L1点的物理意义及其在空间任务中的应用价值：
- L1点为什么能保持卫星相对于地球和月球静止？

_L1点保持卫星相对静止的物理机制
L1点位于地球与月球的连线上，是地月系统拉格朗日点之一。其稳定性来源于引力与离心力的动态平衡：

引力平衡：地球和月球对卫星的引力合力恰好提供绕地球公转所需的向心力。在此点，卫星与月球同步绕地球运动，周期相同，因此能保持相对位置固定259。

动力学特性：根据限制性三体问题，L1点处的小物体在引力作用下无需额外轨道调整即可维持稳定，极大节省燃料14。
L1点的实际应用价值
深空探测与观测

L1点不受地球或月球遮挡，可实现对太阳或地球向阳面的持续观测。例如，日地L1点被用于太阳活动监测（如我国嫦娥五号轨道器拓展任务）468。

地月L1点可作为月球背面探测的中继站，解决通信遮挡问题（类似鹊桥卫星在L2点的作用）29。

低能耗轨道与物资中转

由于轨道稳定性，L1点附近可部署长期驻留的航天器（如太空基地或物资中转站），减少燃料消耗，并作为深空探测的跳板136。

### 4.2 数值方法的比较

讨论牛顿法和弦截法在求解此问题时的优缺点：
- 收敛速度比较
- 1. 收敛速度
牛顿法：
具有二次收敛速度，在初始值足够接近真实解时，迭代次数少且收敛极快。例如在用户代码中，牛顿法通常能在5-10次迭代内达到收敛。但对函数光滑性要求较高，需导数连续且非零。

弦截法：
收敛速度为超线性（≈1.618阶），虽快于线性但慢于牛顿法。实际测试中，用户代码的弦截法需约10-15次迭代收敛，耗时略高于牛顿法。
- 对初值/初始区间的敏感性
- 2. 初值敏感性
牛顿法：
高度依赖初始猜测的准确性。若初值偏离真实解较远（如误差超过地月距离的5%），可能因导数趋零导致发散。例如，若初始猜测设为3.0e8 m（远小于L1实际值3.264e8 m），迭代可能失败。

弦截法：
依赖初始区间的包围性而非精确初值。用户代码中区间[3.2e8,3.7e8] m有效覆盖L1点，即使区间跨度达15%（约5e7 m）仍能收敛。但对函数在区间内的单调性要求严格，若区间内存在多个根或极值点可能失效。
- 计算复杂度和实现难度
. 计算复杂度与实现难度
牛顿法：
复杂度较高，需显式计算导数函数。例如用户代码中需额外实现lagrange_equation_derivative，增加编码与调试成本。单次迭代需计算函数值及导数值各一次，但若导数解析式复杂（如涉及高阶项），计算量显著上升。

弦截法：
实现简单，仅需函数求值。代码中仅需传递函数lagrange_equation，无需导数推导。单次迭代需两次函数求值，但若函数本身计算耗时（如含多次幂运算），总耗时可能接近牛顿法。

## 5. 结论

总结本实验的主要发现和收获，包括对拉格朗日点物理意义的理解以及数值方法在天体力学问题中的应用。

_其他拉格朗日点（L2-L5）的确定与特点
L2-L5点的位置与特性：

L2点：位于月球外侧（远离地球侧），满足地球引力+月球引力=离心力。方程形式与L1类似，但方向相反。常用于深空观测（如放置望远镜），例如地月L2点可监测月球背面。

L3点：位于地球背对月球侧，受地球引力与系统离心力平衡。实际应用较少，因稳定性差且易受太阳引力干扰。

L4/L5点：位于月球轨道前后60°处，与地月构成等边三角形。稳定性高（受科里奥利力与势能谷保护），可自然聚集天体（如木星特洛伊小行星）。地月L4/L5点适合长期驻留任务或空间站建设。

确定方法：

L1-L3：通过共线力平衡方程数值求解（如牛顿法）。

L4/L5：解析解为等边三角形顶点，直接由几何关系确定，无需迭代。）_

## 核心代码片段

请粘贴您实现的关键函数代码（如L1点位置方程和牛顿法实现），并简要说明代码的核心思路。

```x = x0
    iterations = 0
    converged = False

    for i in range(max_iter):
        fx = f(x)
        if abs(fx) < tol:
            converged = True
            iterations = i + 1
            break

        dfx = df(x)
        if abs(dfx) < 1e-14:  # 避免除以接近零的数
            break

        delta = fx / dfx
        x_new = x - delta

        # 检查相对变化是否小于容差
        if abs(delta / x) < tol:
            converged = True
            iterations = i + 1
            x = x_new
            break

        x = x_new
        iterations = i + 1

    return x, iterations, converged
```
